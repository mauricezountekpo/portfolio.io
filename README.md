## 🛠 Stack
- **Core:** Vue 3 (Composition API)
- **Build:** Vite 8 (Ultra-fast HMR)
- **Motion:** IntersectionObserver API (Native Scroll Animations)
- **Typography:** Inter (Premium Variable Font)

## 🚀 Setup
```bash
# Clone and install
git clone https://github.com/mauricezountekpo/portfolio.git

npm install

# Run development

npm run dev
```
